#	General notes:
#
#	This table provides an approximate mapping between the glyph codes
#	of the TeX Computer Modern Math Italic font and Unicode.
#
#	The glyph codes do not distinguish between characters that happen
#	to have the same appearance, so this table is only informative.
#	For example, 0x41 is equated with LATIN CAPITAL LETTER A, although
#	it is also used within TeX to render GREEK CAPITAL LETTER ALPHA.
#
#	The TeX accents are zero-width, and are represented here by the
#	corresponding Unicode non-spacing marks.  However, a straight
#	translation from TeX codes to Unicode will produce incorrect Unicode,
#	as non-spacing marks in Unicode must follow the character they
#	combine with.  This problem cannot be solved by a mapping table.
#
#	Format:  Three tab-separated columns
#		 Column #1 is the TeX font code (in hex as 0xXX)
#		 Column #2 is the Unicode (in hex as 0xXXXX)
#		 Column #3 the Unicode name (follows a comment sign, '#')
#
#	The entries are in TeX font order
#
